# Azure-Sentinel-SOC-Detection-and-Response-Project

## Goal
Build a SOC environment to simulate attacks, create detection rules, enrich alerts, and investigate alerts.

## Planned Components
- Azure VM (target)
- Kali VM (attacker)
- Microsoft Sentinel (SIEM)
- Detection rules (KQL)
- Python alert enrichment
- Attack chain correlation (MITRE ATT&CK)

## Planned Detections
- Brute force login attempts (Detect multiple failed login attempts)
- Suspicious PowerShell activity (Identify unusual PowerShell commands)
- Unusual Process Execution (Detect anomalous processes)
- Impossible Travel (Identify logins from geographically distant locations in a short amount of time)
